# mega

Go implementation of the MEGA client used by this repository.

## Status

This package is the beginning of the migration from `megajs` to Go. It currently includes:

- MEGA URL parsing for old and new public link formats
- Core crypto primitives and helpers needed for protocol parity
- Foundational API/Storage/File types for the rest of the port

## Quick Start

```bash
go test ./...
```
